# 문제 해결

화면에 표시된 메시지와 가장 가까운 항목부터 확인하세요.

## KREAM 로그인이 필요합니다

오른쪽 위 사용자 메뉴에서 `KREAM 로그인`을 선택하고 Edge 창에서 로그인을 완료합니다. 이미 로그인했는데 계속 같은 메시지가 나오면 앱에서 로그아웃한 뒤 다시 로그인하세요.

## 로그인 후 Edge 창이 닫히지 않습니다

KREAM 홈 또는 마이페이지가 정상적으로 표시되는지 확인합니다. 추가 인증이나 약관 동의 화면이 남아 있으면 완료하세요. 앱이 로그인 상태를 확인하면 창이 자동으로 닫힙니다.

## 상품 검색이 오래 걸리거나 시간 초과됩니다

1. 인터넷 연결을 확인합니다.
2. KREAM 로그인이 유지되는지 확인합니다.
3. 잠시 기다린 뒤 모델번호로 다시 검색합니다.
4. 반복되면 앱을 재시작합니다.

검색 결과 응답을 놓친 경우에도 앱이 현재 화면에서 결과를 읽어 복구를 시도합니다.

## KREAM 요청 제한이 감지되었습니다

짧은 시간에 확인 요청이 집중되면 앱이 감지 간격을 자동으로 늘립니다. 대상을 반복해서 중지하고 시작하지 말고 다음 확인 시각까지 기다리세요. 제한이 계속되면 모든 감지를 중지하고 충분한 시간이 지난 뒤 다시 시작합니다.

## 신청 결과를 확인할 수 없습니다

자동 재결제를 막기 위해 해당 감지가 중지됩니다. KREAM 공식 보관판매 신청 내역과 결제 내역을 먼저 확인한 다음 필요한 경우에만 다시 시작하세요.

## Microsoft Edge를 열 수 없습니다

Windows 업데이트에서 Microsoft Edge가 정상 설치되어 있는지 확인하고 Edge를 한 번 직접 실행한 뒤 앱을 다시 시작합니다.

## 업데이트가 설치되지 않습니다

실행 중인 감지를 중지하고 앱을 관리자 권한으로 다시 실행합니다. 계속 실패하면 최신 설치 파일을 공식 GitHub Release에서 내려받아 기존 설치 위에 설치합니다.

## 문의하기

해결되지 않는 문제는 [GitHub @jwpgdx](https://github.com/jwpgdx)의 공식 저장소 문의 기능을 이용하세요. 공개 글에는 KREAM 아이디, 비밀번호, 쿠키, 결제 정보, 배송지 또는 개인정보가 보이는 화면을 올리지 마세요.
