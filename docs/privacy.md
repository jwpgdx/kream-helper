# 개인정보 처리방침

- 시행일: 2026년 7월 17일
- 처리 주체: `jwpgdx`
- 문의: [GitHub @jwpgdx](https://github.com/jwpgdx)

설치 프로그램과 함께 관리하는 전체 문서는 [개인정보 처리방침 원문](./legal/privacy-ko.txt)에서 확인할 수 있습니다.

KREAM Helper는 이용자의 KREAM 로그인 정보와 감지 데이터를 제작자 서버에 모으지 않는 것을 원칙으로 합니다.

## 제작자 서버에서 처리하는 정보

| 항목 | 목적 | 저장 및 보유 |
| --- | --- | --- |
| 운영체제 종류, 앱 버전 | 서버 상태, 기능 설정, 호환 버전 및 업데이트 확인 | 제작자 KV/D1 데이터베이스에 이용자별로 저장하지 않음 |
| 접속 IP, 요청 시각, 요청 URL과 통신 메타데이터 | 서버 전송, 보안, 장애 분석 | Cloudflare Workers 호출 로그에 현재 Free 플랜 기준 3일, 제공자 사양상 최대 7일 |

KREAM 아이디, 비밀번호, 로그인 쿠키, 인증 헤더, 감지 상품, 감지 기록, 배송지, 결제수단과 결제 정보는 제작자 서버로 전송하지 않습니다. 개인정보를 판매하거나 맞춤형 광고에 사용하지 않습니다.

## 이용자 컴퓨터에 저장하는 정보

| 저장 종류 | 포함될 수 있는 내용 | 삭제 방법 |
| --- | --- | --- |
| 전용 Edge 프로필 | KREAM 쿠키, 로그인 세션, 브라우저 캐시 | 앱의 `로그아웃` 실행 |
| 앱 로컬 저장소 | 상품, 모델번호, 사이즈, 수량, 주기, 자동 신청 설정 | 앱에서 감지 상품 삭제 |
| 감지 기록 | 확인 시각, 결과, 상품과 메시지 | 감지 기록 화면에서 기록 지우기 |
| 앱 설정 | 시작 프로그램, 알림과 종료 방식 | 전체 앱 데이터 폴더 삭제 |
| 진단 화면 캡처 | 캡처 기능을 실행한 경우 현재 KREAM 화면 | 해당 파일 또는 전체 데이터 폴더 삭제 |

로컬 데이터는 Windows 사용자별 `%APPDATA%\kream-helper` 폴더와 Electron 로컬 저장소에 보관됩니다. 프로그램 제거만으로는 삭제되지 않습니다.

## KREAM으로 직접 전송되는 정보

상품 검색, 로그인 확인, 보관판매 상태 확인과 신청을 위한 요청은 이용자의 컴퓨터에서 KREAM으로 직접 전송됩니다. KREAM의 이용약관과 개인정보 처리방침이 적용되며 제작자는 이용자의 KREAM 비밀번호나 결제수단을 수신하지 않습니다.

## 처리위탁 및 국외 이전

| 이전받는 자 | 이전 국가 | 시기 및 방법 | 이전 항목 | 목적 | 보유 기간 |
| --- | --- | --- | --- | --- | --- |
| Cloudflare, Inc. | 주로 미국 및 유럽경제지역, 글로벌 운영 국가 | 앱 실행, 서버 및 업데이트 확인 시 HTTPS 전송 | 운영체제 종류, 앱 버전, 접속 IP와 통신 메타데이터 | Workers/R2 전송, 보안, 장애 분석, 업데이트 | Workers 호출 로그는 현재 3일, 제공자 사양상 최대 7일 |

Cloudflare 연락처는 `privacyquestions@cloudflare.com`입니다. 이용자는 설치를 취소하거나 방화벽에서 서버 통신을 차단해 처리를 거부할 수 있으며, 이 경우 원격 기능 설정과 자동 업데이트가 동작하지 않을 수 있습니다.

자세한 내용은 [Cloudflare 개인정보 처리방침](https://www.cloudflare.com/policies/privacy/)을 확인하세요. GitHub 배포 또는 문의 페이지를 직접 방문하면 GitHub가 접속 정보를 자체 정책에 따라 처리합니다.

## 제3자 제공과 광고

제작자는 이용자의 개인정보를 광고주나 제휴사에 판매 또는 제공하지 않습니다. 현재 버전에는 광고, 제휴 쿠키 삽입, 쇼핑몰 리디렉션 또는 제휴 프로그램 묶음 설치가 없습니다.

## 이용자의 권리와 문의

이용자는 앱에서 KREAM 로그아웃, 감지 상품 삭제, 기록 지우기를 실행하고 전체 앱 데이터 폴더를 삭제할 수 있습니다. 개인정보 처리 문의는 [GitHub @jwpgdx](https://github.com/jwpgdx)의 연락 수단을 이용하세요.

공개 GitHub 문의에는 KREAM 아이디, 비밀번호, 쿠키, 결제 정보, 배송지 또는 식별 가능한 화면 캡처를 게시하지 마세요.

중요하거나 이용자에게 불리한 변경은 원칙적으로 시행 30일 전, 그 밖의 변경은 7일 전부터 공식 GitHub 배포 저장소에 알립니다.
