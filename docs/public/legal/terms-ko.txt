KREAM Helper 이용약관 및 개인정보 안내

버전: 1.0
시행일: 2026년 7월 17일
제작자: jwpgdx
문의: GitHub @jwpgdx (https://github.com/jwpgdx)

[중요 사항 요약]
1. KREAM Helper는 크림 주식회사가 제작, 승인 또는 제휴한 공식 프로그램이 아닙니다.
2. KREAM은 매크로를 통한 과도한 API 호출 등 서비스 운영을 방해하는 행위에 대해 계정 이용을 제한할 수 있다고 안내합니다. 이 프로그램의 사용이 KREAM 정책상 허용된다고 보장되지 않습니다.
3. 자동 신청을 켜면 신청 가능 상태가 확인되는 즉시 사용자가 지정한 모델, 사이즈와 수량으로 필수 항목에 동의하고 보증금을 결제할 수 있습니다.
4. KREAM 로그인 세션과 감지 설정 및 기록은 이 컴퓨터에 저장됩니다. 프로그램만 제거해도 로컬 데이터가 자동으로 삭제되지는 않습니다.
5. 서버 설정 및 업데이트 확인을 위해 앱 버전과 운영체제 종류가 Cloudflare 기반 서버로 전송되며, 접속 IP 등 통신 정보는 Cloudflare에서 처리될 수 있습니다. KREAM 아이디, 비밀번호, 세션, 감지 상품, 결제 정보와 감지 기록은 제작자 서버로 전송하지 않습니다.

위 사항과 아래 약관에 동의하지 않으면 설치를 취소하고 프로그램을 사용하지 마십시오.

제1조 (목적 및 동의)
이 약관은 KREAM Helper(이하 "프로그램")의 이용 조건, 제작자와 이용자의 권리 및 책임, 로컬 데이터와 네트워크 통신에 관한 사항을 정합니다. 이용자가 설치 화면에서 "동의함"을 선택하거나 프로그램을 사용하면 이 약관에 동의한 것으로 봅니다.

제2조 (비공식 프로그램 및 권리 관계)
1. 프로그램은 KREAM 보관판매 신청 가능 상태 확인과 신청 절차를 돕는 독립적인 제3자 도구입니다.
2. 프로그램과 제작자는 크림 주식회사와 제휴, 협력, 후원 또는 공식 승인 관계가 없습니다.
3. KREAM 명칭, 상표, 서비스 화면, 상품 이미지와 상품 정보에 관한 권리는 각 권리자에게 있습니다. 프로그램에서의 표시는 호환 대상과 상품을 식별하기 위한 것입니다.
4. 프로그램의 명칭이나 화면만으로 KREAM의 공식 프로그램 또는 제휴 프로그램이라고 오인해서는 안 됩니다.

제3조 (제공 기능)
1. 이용자가 지정한 상품, 사이즈와 수량의 KREAM 보관판매 신청 가능 상태를 설정된 간격으로 확인합니다.
2. 자동 신청이 꺼져 있으면 신청 가능 상태를 알리고 KREAM 신청 화면을 열 수 있도록 돕습니다.
3. 자동 신청이 켜져 있으면 이용자의 사전 확인에 따라 KREAM 화면의 필수 동의 항목을 선택하고 보증금 결제를 포함한 신청을 한 차례 수행할 수 있습니다.
4. 서버 상태, 기능 설정과 새 버전을 확인하고, 이용자가 승인하면 업데이트를 설치할 수 있습니다.
5. KREAM, Microsoft Edge, 운영체제 또는 네트워크 환경의 변경에 따라 일부 기능이 지연, 변경 또는 중단될 수 있습니다.

제4조 (KREAM 계정 및 정책 준수)
1. 이용자는 본인 명의의 KREAM 계정과 본인이 사용할 권한이 있는 결제수단만 사용해야 합니다.
2. 이용자는 KREAM의 최신 이용약관, 운영정책, 보관판매 조건, 페널티 및 취소 조건을 직접 확인하고 준수해야 합니다.
3. KREAM은 매크로를 통한 과도한 API 호출 등 서비스의 정상 운영을 방해하는 행위를 이용 제한 사유로 안내하고 있습니다. 프로그램은 최소 확인 간격, 순차 확인, 지연 및 재시도 제한을 적용하지만, 이러한 조치가 KREAM의 허용 또는 계정 안전을 보장하지 않습니다.
4. KREAM이 프로그램 사용을 금지하거나 중단을 요청하는 경우 이용자는 즉시 감지를 중지하고 프로그램 사용을 중단해야 합니다.
5. 이용자의 KREAM 약관 또는 법령 위반으로 발생한 계정 제한, 혜택 소멸, 거래 취소, 페널티 등은 제작자의 귀책이 없는 범위에서 이용자에게 책임이 있습니다.

제5조 (자동 신청 및 금전 결제)
1. 자동 신청은 기본적으로 꺼져 있으며, 이용자가 감지 대상마다 직접 켜야 합니다.
2. 프로그램은 자동 신청 대상을 추가하거나 감지를 시작할 때 상품명, 모델번호, 사이즈, 수량과 예상 보증금을 표시하고 이용자의 별도 확인을 받습니다.
3. 자동 신청이 시작되면 이용자가 자리를 비운 동안에도 KREAM에 저장된 결제수단을 통해 보증금이 결제될 수 있습니다. 예상 보증금은 현재 프로그램 기준 수량 1개당 3,000원으로 계산하지만, 실제 금액과 조건은 결제 직전 KREAM 화면의 표시가 우선합니다.
4. 이용자는 시작 전에 상품, 모델번호, 사이즈, 수량, 실제 보증금, 결제수단, 발송 가능 수량과 기한을 확인해야 합니다. 특히 큰 수량을 지정하면 보증금과 이행 부담이 크게 증가할 수 있습니다.
5. 프로그램은 결제 이후 신청 결과를 명확히 확인하지 못하면 자동 재결제를 시도하지 않고 해당 감지를 중지하도록 설계되어 있습니다. 다만 네트워크 단절이나 KREAM 화면 변경으로 결과가 불명확할 수 있으므로 이용자는 KREAM 계정에서 신청 및 결제 내역을 즉시 확인해야 합니다.
6. 신청 취소, 환불, 보증금 반환, 발송 지연과 페널티에는 KREAM이 실제로 고지한 조건이 적용됩니다.

제6조 (금지 행위)
이용자는 다음 행위를 해서는 안 됩니다.
1. 타인의 계정, 명의, 결제수단 또는 세션을 무단으로 사용하는 행위
2. 확인 간격이나 보호장치를 변조하여 과도한 요청을 보내거나 KREAM 서비스 운영에 부하 또는 장애를 일으키는 행위
3. CAPTCHA, 접근 제한, 인증, 요청 서명 또는 그 밖의 기술적 보호조치를 우회하는 행위
4. 프로그램을 이용해 허위 신청, 시세 조작, 부정 거래, 재판매 대행 또는 관계 법령이나 KREAM 정책에 위반되는 행위를 하는 것
5. 악성코드 삽입, 변조 배포, 제작자 또는 KREAM의 공식 프로그램으로 사칭하는 행위

제7조 (로그인 및 로컬 데이터)
1. 이용자는 프로그램이 실행한 Microsoft Edge의 KREAM 또는 소셜 로그인 화면에서 직접 로그인합니다. 프로그램은 KREAM 아이디나 비밀번호를 별도 입력받거나 제작자 서버로 전송하지 않습니다.
2. KREAM 쿠키와 로그인 세션, 브라우저 캐시 등은 각 Windows 사용자 계정의 전용 Edge 프로필에 저장됩니다.
3. 감지 대상에는 모델번호, 사이즈, 수량, 확인 간격, 자동 신청 설정, 상품 식별자, 상품명, 이미지 주소와 카테고리 등이 저장될 수 있습니다.
4. 감지 기록에는 확인 시각, 결과 상태, 상품명, 모델번호, 사이즈, 시도 횟수와 결과 메시지가 저장될 수 있습니다. 앱 설정에는 시작 프로그램, 알림과 종료 방식 설정이 저장될 수 있습니다.
5. 위 데이터는 원칙적으로 %APPDATA%\kream-helper 폴더와 프로그램의 로컬 저장소에만 보관됩니다. 진단용 화면 캡처 기능을 실행한 경우 캡처도 같은 폴더 아래에 로컬로 저장되며 자동 전송되지 않습니다.
6. 프로그램의 "로그아웃"은 전용 Edge 프로필을 삭제합니다. 감지 상품은 각 상품의 삭제 기능으로, 감지 기록은 기록 지우기 기능으로 삭제할 수 있습니다.
7. 업데이트와 재설치를 위해 프로그램 제거 후에도 로컬 데이터가 유지됩니다. 완전히 삭제하려면 프로그램과 전용 Edge 창을 종료한 뒤 %APPDATA%\kream-helper 폴더를 직접 삭제해야 합니다.

제8조 (서버 통신 및 개인정보 처리)
1. 프로그램은 실행 또는 서버 확인 시 운영체제 종류와 현재 앱 버전을 Cloudflare Workers 서버에 HTTPS로 전송합니다. 제작자는 이 값을 이용자별 데이터베이스에 저장하지 않습니다.
2. 서버 및 업데이트 파일 전송 과정에서 Cloudflare는 접속 IP, 요청 시각, 요청 URL, 트래픽 라우팅 정보와 시스템 구성 정보 등 통신 메타데이터를 처리할 수 있습니다. 현재 Cloudflare Workers 호출 로그는 Free 플랜에서 3일간 보관되며, 제공자 정책 또는 플랜 변경 시 현재 문서 기준 최대 7일까지 보관될 수 있습니다.
3. Cloudflare는 미국 소재의 글로벌 사업자로서 관련 정보를 주로 미국 및 유럽경제지역에 저장하고 전 세계 운영 거점에서 처리할 수 있습니다. 이전은 프로그램 실행, 서버 확인 또는 업데이트 확인 시 암호화된 통신으로 이루어지며, 서버 운영, 보안, 장애 분석과 업데이트 제공을 목적으로 합니다.
4. 이용자는 설치를 취소하거나 프로그램의 서버 주소를 방화벽에서 차단하여 위 처리를 거부할 수 있습니다. 이 경우 서버 상태 확인, 원격 기능 설정과 자동 업데이트가 동작하지 않을 수 있습니다.
5. KREAM 아이디, 비밀번호, 로그인 쿠키, 인증 헤더, 감지 상품, 활동 기록, 배송지, 결제수단과 결제 정보는 제작자 서버로 전송하지 않습니다. KREAM 기능 수행을 위한 요청은 이용자의 컴퓨터에서 KREAM으로 직접 전송되며 KREAM의 약관과 개인정보 처리방침이 적용됩니다.
6. 제작자는 개인정보를 판매하거나 맞춤형 광고 목적으로 이용하지 않으며, 현재 버전에는 광고, 제휴 리디렉션, 제휴 쿠키 삽입 또는 제휴 프로그램 묶음 설치가 포함되어 있지 않습니다.
7. 자세한 처리 내용과 삭제 방법은 공식 배포 저장소의 "개인정보 처리방침"에서 확인할 수 있습니다.

제9조 (프로그램 변경 및 업데이트)
1. 제작자는 보안, KREAM 정책, 기술 환경 또는 운영상 필요에 따라 프로그램 기능을 변경하거나 제공을 중단할 수 있습니다.
2. 기능 유지에 필요한 보안 또는 호환성 업데이트가 제공될 수 있습니다. 설치 전 프로그램을 종료하고 진행 중인 신청 결과를 확인해야 합니다.
3. 이용자에게 중요하거나 불리한 약관 변경은 원칙적으로 시행 30일 전, 그 밖의 변경은 7일 전부터 GitHub 공식 배포 저장소에 공지합니다. 긴급한 법령, 보안 또는 제3자 서비스 변경은 사유를 밝히고 더 짧은 기간으로 적용할 수 있습니다.
4. 변경된 약관에 동의하지 않으면 업데이트 또는 사용을 중단하고 프로그램을 제거할 수 있습니다. 별도 동의가 필요한 변경은 업데이트 설치 과정이나 프로그램 화면에서 다시 동의를 받습니다.

제10조 (제휴 프로그램 및 광고)
1. 현재 버전에는 제휴 프로그램과 광고가 포함되어 있지 않습니다.
2. 향후 제휴 프로그램 또는 광고가 추가되더라도 제공자, 프로그램명, 주요 기능, 데이터 처리, 설치 여부와 경제적 대가를 별도 화면에서 명확히 알리고 개별 선택과 동의를 받아야 합니다.
3. 기존 약관 동의만으로 제휴 프로그램을 자동 설치하거나, 브라우저 접속을 몰래 가로채거나, 이용자의 명시적 클릭 없이 제휴 ID 또는 쿠키를 삽입하지 않습니다.

제11조 (보증의 범위)
1. 프로그램은 무료로 현재 상태 그대로 제공됩니다. 제작자는 KREAM의 정책상 허용, 중단 없는 작동, 모든 신청 가능 상태의 감지, 신청 성공, 취소 성공, 계정 유지 또는 특정한 경제적 결과를 보증하지 않습니다.
2. KREAM, Microsoft Edge, Cloudflare, 결제수단, 운영체제, 통신망 등 제3자 서비스의 변경 또는 장애는 제작자가 통제할 수 없습니다.
3. 이용자는 중요한 신청 및 결제 결과를 KREAM 공식 화면에서 직접 확인하고, 오류가 의심되면 즉시 감지를 중지해야 합니다.

제12조 (책임)
1. 제작자는 프로그램의 결함 등 제작자에게 책임 있는 사유로 이용자에게 손해가 발생한 경우 관계 법령에 따라 책임을 부담합니다.
2. 이용자의 설정 오류, 확인 의무 위반, 금지 행위, KREAM 정책 위반 또는 제작자에게 책임이 없는 제3자 서비스 장애로 발생한 손해는 제작자의 귀책이 없는 범위에서 이용자가 부담합니다.
3. 관계 법령이 허용하는 범위에서 제작자의 경미한 과실로 인한 손해배상은 해당 과실과 상당인과관계가 있는 통상적이고 직접적인 손해를 기준으로 합니다.
