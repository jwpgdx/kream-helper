# 개인정보 처리방침

- 시행일: 2026년 7월 17일
- 처리 주체: `jwpgdx`
- 문의: [GitHub @jwpgdx](https://github.com/jwpgdx)

KREAM Helper는 이용자의 KREAM 로그인 정보와 감지 데이터를 제작자 서버에 모으지 않는 것을 원칙으로 합니다. 이 문서는 컴퓨터에 저장되는 데이터와 서버 확인 과정에서 처리되는 정보를 구분해 설명합니다.

## 1. 처리하는 정보와 목적

### 제작자 서버에서 처리하는 정보

| 항목 | 목적 | 저장 및 보유 |
| --- | --- | --- |
| 운영체제 종류, 앱 버전 | 서버 상태, 기능 설정, 호환 버전 및 업데이트 확인 | 제작자 KV/D1 데이터베이스에 이용자별로 저장하지 않음 |
| 접속 IP, 요청 시각, 요청 URL, 트래픽 라우팅 및 시스템 구성 정보 등 통신 메타데이터 | 서버 전송, 보안, 장애 분석 | Cloudflare Workers 호출 로그에 현재 Free 플랜 기준 3일, 제공자 사양상 최대 7일 |

KREAM 아이디, 비밀번호, 로그인 쿠키, 인증 헤더, 감지 상품, 감지 기록, 배송지, 결제수단과 결제 정보는 제작자 서버로 전송하지 않습니다. 개인정보를 판매하거나 맞춤형 광고에 사용하지 않습니다.

### 이용자 컴퓨터에만 저장하는 정보

| 저장 위치/종류 | 포함될 수 있는 내용 | 삭제 방법 |
| --- | --- | --- |
| 전용 Edge 프로필 | KREAM 쿠키, 로그인 세션, 브라우저 캐시와 KREAM이 제공한 프로필 표시 정보 | 앱의 `로그아웃` 실행 |
| 앱 로컬 저장소 | 모델번호, 사이즈, 수량, 확인 간격, 자동 신청 여부, 상품 식별자, 상품명, 이미지 주소, 카테고리 | 앱에서 감지 상품 삭제 |
| `monitor-activity.json` | 확인 시각, 결과 상태, 상품명, 모델번호, 사이즈, 시도 횟수, 결과 메시지 | 감지 기록 화면에서 기록 지우기 |
| `app-settings.json` | 시작 프로그램, 알림, 종료 방식 설정 | 전체 앱 데이터 폴더 삭제 |
| `screenshots` 폴더 | 진단용 화면 캡처 기능을 실행한 경우 현재 KREAM 화면 | 해당 파일 또는 전체 앱 데이터 폴더 삭제 |

로컬 데이터는 Windows 사용자별 `%APPDATA%\kream-helper` 폴더와 Electron 로컬 저장소에 보관됩니다. 업데이트와 재설치를 위해 프로그램 제거만으로는 삭제되지 않습니다. 완전 삭제는 프로그램과 전용 Edge 창을 종료한 뒤 해당 폴더를 삭제하면 됩니다.

## 2. KREAM으로 직접 전송되는 정보

상품 검색, 로그인 상태 확인, 보관판매 상태 확인과 신청을 위한 요청은 이용자의 컴퓨터에서 KREAM으로 직접 전송됩니다. 해당 처리에는 KREAM의 이용약관과 개인정보 처리방침이 적용됩니다. 제작자는 이용자의 KREAM 계정 비밀번호나 결제수단을 수신하지 않습니다.

## 3. 처리위탁 및 국외 이전

서버 설정과 업데이트 제공을 위해 다음 해외 서비스를 사용합니다.

| 이전받는 자 | 연락처 | 이전 국가 | 시기 및 방법 | 이전 항목 | 목적 | 보유 기간 |
| --- | --- | --- | --- | --- | --- | --- |
| Cloudflare, Inc. | `privacyquestions@cloudflare.com` | 주로 미국 및 유럽경제지역, Cloudflare 글로벌 운영 국가 | 앱 실행, 서버 확인 및 업데이트 확인 시 HTTPS 전송 | 운영체제 종류, 앱 버전, 접속 IP와 통신 메타데이터 | Workers/R2 전송, 서버 보안, 장애 분석, 업데이트 제공 | Workers 호출 로그는 현재 3일, 제공자 사양상 최대 7일. 그 밖의 처리는 Cloudflare 정책상 목적 달성에 필요한 기간 |

이 처리는 프로그램의 원격 설정과 업데이트 제공에 필요한 위탁 및 보관으로서 개인정보 처리방침에 공개합니다. 이용자는 설치를 취소하거나 방화벽에서 서버 통신을 차단해 국외 이전을 거부할 수 있습니다. 이 경우 원격 기능 설정, 서버 상태 확인과 자동 업데이트가 동작하지 않을 수 있습니다.

Cloudflare는 미국 소재의 글로벌 사업자이며, 이용자 통신 정보를 자사 개인정보 처리방침에 따라 처리합니다. 자세한 내용은 [Cloudflare Privacy Policy](https://www.cloudflare.com/policies/privacy/)에서 확인할 수 있습니다.

이용자가 GitHub 배포 페이지나 문의 페이지를 직접 방문하면 GitHub가 접속 정보를 자체 정책에 따라 처리합니다. 프로그램은 KREAM 세션이나 감지 데이터를 GitHub로 자동 전송하지 않습니다.

## 4. 제3자 제공

제작자는 이용자의 개인정보를 광고주, 제휴사 또는 그 밖의 제3자에게 판매하거나 제공하지 않습니다. 법령상 의무가 있거나 이용자가 별도로 동의한 경우는 예외로 합니다. 현재 버전에는 광고, 제휴 쿠키 삽입, 쇼핑몰 리디렉션 또는 제휴 프로그램 묶음 설치가 없습니다.

## 5. 안전성 확보 조치

- 제작자 서버와의 통신은 HTTPS를 사용합니다.
- KREAM 비밀번호는 프로그램 자체 입력창이나 제작자 서버에서 수집하지 않습니다.
- KREAM 세션과 감지 데이터는 이용자의 Windows 계정 아래 로컬에 분리해 저장합니다.
- 감지 기록에서 비밀번호, 쿠키, 인증 헤더, 결제 및 배송 정보로 추정되는 필드는 기록하지 않도록 제한합니다.
- 서버 데이터베이스에는 이용자별 사용 내역이나 감지 대상을 생성하지 않습니다.

## 6. 이용자의 권리와 문의

이용자는 앱에서 KREAM 로그아웃, 감지 상품 삭제, 기록 지우기를 실행할 수 있고 전체 앱 데이터 폴더를 삭제할 수 있습니다. 서버 통신 또는 개인정보 처리에 관한 문의는 [GitHub @jwpgdx](https://github.com/jwpgdx)의 연락 수단을 이용하세요.

공개 GitHub 이슈에는 KREAM 아이디, 비밀번호, 쿠키, 결제 정보, 배송지 또는 식별 가능한 화면 캡처를 게시하지 마세요. 권리 요청에 본인 확인이 필요하거나 보유 중인 정보만으로 특정 이용자를 식별할 수 없는 경우 추가 확인을 요청할 수 있습니다.

## 7. 변경

중요하거나 이용자에게 불리한 변경은 원칙적으로 시행 30일 전, 그 밖의 변경은 7일 전부터 공식 GitHub 배포 저장소에 알립니다. 법령, 보안 또는 외부 서비스의 긴급한 변경은 사유를 밝히고 더 짧은 기간으로 적용할 수 있습니다.
