# 빠른 시작

처음 설치한 뒤 아래 순서대로 진행하면 첫 상품 감지를 시작할 수 있습니다.

## 1. 프로그램 설치

[GitHub Releases](https://github.com/jwpgdx/kream-helper/releases)에서 최신 Windows 설치 파일을 내려받아 실행합니다. 설치 과정에서 이용약관과 개인정보 안내를 확인한 뒤 계속합니다.

::: warning Windows 게시자 경고
현재 설치 파일은 코드 서명 전이므로 Windows에서 `알 수 없는 게시자`로 표시될 수 있습니다. 파일 이름과 공식 배포 주소, SHA-256 체크섬을 확인한 뒤 설치하세요.
:::

## 2. KREAM 로그인

앱 오른쪽 위 사용자 메뉴에서 `KREAM 로그인`을 선택합니다. Microsoft Edge 창이 열리면 KREAM 또는 소셜 계정으로 직접 로그인합니다. 연결이 확인되면 Edge 창은 자동으로 닫힙니다.

## 3. 감지 상품 추가

`감지 상품` 화면에서 `추가`를 누르고 상품명 또는 모델번호를 검색합니다. 상품을 선택한 다음 사이즈, 신청 수량과 감지 주기를 설정합니다.

## 4. 자동 신청 여부 확인

처음에는 `신청 가능 시 자동 신청`을 끈 상태로 사용해 감지 흐름을 확인하는 것을 권장합니다. 자동 신청을 켜면 신청 가능 상태가 확인되는 즉시 필수 항목에 동의하고 보증금을 결제할 수 있습니다.

## 5. 감지 시작

상품을 저장한 뒤 각 행의 `감지 시작`을 누릅니다. 여러 상품을 한꺼번에 시작하려면 화면 위의 `전체 시작`을 사용합니다.
