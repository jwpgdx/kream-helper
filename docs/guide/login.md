# KREAM 로그인

KREAM Helper는 앱 내부에 아이디와 비밀번호를 입력받지 않습니다. 로그인 버튼을 누르면 전용 Microsoft Edge 창이 열리고, 사용자가 KREAM 또는 소셜 로그인 화면에서 직접 로그인합니다.

## 로그인하기

1. 오른쪽 위 사용자 아이콘을 누릅니다.
2. `KREAM 로그인`을 선택합니다.
3. 새 Edge 창에서 원하는 로그인 방식을 선택합니다.
4. KREAM 로그인이 확인될 때까지 기다립니다.
5. 연결이 완료되면 Edge 창이 닫히고 앱 화면이 갱신됩니다.

로그인 확인은 사용자가 별도 버튼을 누르지 않아도 앱이 자동으로 수행합니다. 앱이 다시 활성화되거나 네트워크가 복구될 때도 상태를 다시 확인합니다.

## 세션 저장 위치

로그인 쿠키와 세션은 일반 Edge 프로필이 아닌 KREAM Helper 전용 로컬 프로필에 저장됩니다. 제작자 서버로 KREAM 아이디, 비밀번호 또는 로그인 쿠키를 보내지 않습니다.

## 로그아웃

사용자 메뉴에서 `로그아웃`을 선택하면 실행 중인 감지를 중지하고 전용 Edge 프로필을 삭제합니다. 감지 상품과 감지 기록은 그대로 유지됩니다.

::: warning 공용 컴퓨터
공용 컴퓨터에서는 사용을 마친 뒤 반드시 앱에서 로그아웃하고, 필요한 경우 전체 로컬 데이터도 삭제하세요.
:::
