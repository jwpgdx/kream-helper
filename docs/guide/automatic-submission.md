# 자동 신청

자동 신청은 신청 가능 상태가 확인되었을 때 KREAM의 필수 동의 항목을 선택하고 보증금을 결제해 신청을 완료하는 기능입니다. **기본값은 꺼짐**입니다.

## 켜기 전에 확인할 항목

<GuideImage
  src="/images/product-settings.png"
  alt="신청 수량과 자동 신청 스위치가 있는 상품 설정 화면"
  caption="자동 신청을 켜기 전에 모델, 사이즈, 수량과 예상 보증금을 확인합니다."
/>

1. 정확한 상품과 모델번호인지 확인합니다.
2. 사이즈 또는 옵션을 확인합니다.
3. 실제로 발송할 수 있는 수량인지 확인합니다.
4. 예상 보증금과 KREAM에 저장된 결제수단을 확인합니다.
5. 발송 기한, 취소와 페널티 조건을 확인합니다.

현재 앱의 예상 보증금은 `수량 × 3,000원`으로 표시됩니다. 실제 금액과 조건은 신청 시점의 KREAM 화면이 우선합니다.

::: danger 자동 결제
자동 신청을 켜고 감지를 시작하면 사용자가 자리를 비운 동안에도 지정한 수량만큼 보증금이 결제될 수 있습니다. 큰 수량을 입력하기 전에 총액을 반드시 확인하세요.
:::

## 신청 결과가 불명확한 경우

결제 이후 신청 완료 여부를 명확하게 확인하지 못하면 앱은 같은 대상을 자동으로 다시 결제하지 않고 감지를 중지합니다. 네트워크가 끊겼거나 화면이 변경된 경우에는 KREAM 공식 보관판매 신청 내역과 결제 내역을 직접 확인하세요.

## 자동 신청 없이 사용하기

자동 신청을 끄면 신청 가능 상태에서 KREAM 신청 화면까지만 열고 멈춥니다. 감지 목록에 나타나는 `신청 화면 열기`를 눌러 내용을 직접 확인하고 신청할 수 있습니다.
