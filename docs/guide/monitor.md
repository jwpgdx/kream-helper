# 감지 시작과 상태

감지 상품은 각각 독립적으로 시작하고 중지할 수 있습니다. 앱은 동시에 무리하게 요청하지 않도록 실제 확인 작업을 순차적으로 처리합니다.

## 시작과 중지

- `감지 시작`: 선택한 상품만 시작합니다.
- `전체 시작`: 중지된 모든 상품을 시작합니다.
- `중지`: 실행 중인 해당 상품을 멈춥니다.
- `전체 중지`: 실행 중인 모든 상품을 멈춥니다.

창을 닫을 때 감지가 실행 중이면 기본 설정에 따라 작업 표시줄 알림 영역에서 계속 실행됩니다.

## 상태 의미

| 상태 | 의미 | 해야 할 일 |
| --- | --- | --- |
| 대기 | 아직 확인하지 않았거나 다음 확인을 기다리는 중 | 다음 확인 시각을 기다립니다. |
| 확인 중 | KREAM 화면에서 현재 상태를 확인하는 중 | 잠시 기다립니다. |
| 신청 제한 | 현재 보관판매 신청이 제한된 상태 | 감지를 계속하거나 중지합니다. |
| 신청 가능 | 보관판매 신청 화면까지 진입한 상태 | 자동 신청이 꺼져 있으면 직접 신청합니다. |
| 신청 완료 | 자동 신청 결과가 명확하게 확인된 상태 | KREAM 공식 화면에서 내역을 다시 확인합니다. |
| 로그인 필요 | 세션이 만료되었거나 로그아웃된 상태 | KREAM에 다시 로그인합니다. |
| 오류 | 요청 제한, 화면 변경 또는 네트워크 문제 | 메시지를 확인하고 재시도합니다. |

KREAM 요청 제한이 감지되면 앱이 다음 확인 간격을 자동으로 늘립니다. CAPTCHA나 보안 절차는 우회하지 않습니다.
