# 감지 기록

`감지 기록` 탭에는 최근 확인 결과와 오류가 시간순으로 저장됩니다. 같은 상품에서 동일한 제한 또는 오류가 반복되면 기록이 한 행으로 묶일 수 있습니다.

## 기록에서 확인할 내용

- 확인 날짜와 시각
- 신청 가능, 오류 또는 중지 결과
- 상품명, 모델번호와 사이즈
- 같은 결과의 반복 횟수
- KREAM 화면에서 확인한 사용자용 메시지

상단 필터로 전체, 신청 가능 완료, 주의 필요 항목을 나눠 볼 수 있습니다. 검색창에서는 상품명과 모델번호를 찾을 수 있습니다.

## 기록 삭제

오른쪽 위 메뉴에서 `기록 지우기`를 선택하면 저장된 감지 기록을 모두 삭제합니다. 이 작업은 감지 상품 설정과 KREAM 로그인 세션에는 영향을 주지 않습니다.

기록은 사용자 컴퓨터에 최대 300건까지 저장되며 제작자 서버로 전송되지 않습니다.
