# 데이터와 삭제

KREAM Helper는 KREAM 세션, 감지 상품, 기록과 앱 설정을 각 Windows 사용자 계정의 로컬 폴더에 저장합니다.

## 저장되는 데이터

| 종류 | 예시 | 제작자 서버 전송 |
| --- | --- | --- |
| 전용 Edge 프로필 | KREAM 쿠키, 로그인 세션과 브라우저 캐시 | 전송하지 않음 |
| 감지 상품 | 상품명, 모델번호, 사이즈, 수량, 주기, 자동 신청 여부 | 전송하지 않음 |
| 감지 기록 | 시각, 상태, 시도 횟수와 결과 메시지 | 전송하지 않음 |
| 앱 설정 | 시작 프로그램, 알림과 종료 방식 | 전송하지 않음 |
| 서버 확인 정보 | 운영체제 종류와 앱 버전 | Cloudflare 서버로 전송 |

## 항목별 삭제

- **KREAM 세션:** 사용자 메뉴에서 `로그아웃`
- **감지 상품:** 상품 오른쪽의 더보기 메뉴에서 `삭제`
- **감지 기록:** 감지 기록 화면 메뉴에서 `기록 지우기`
- **앱 설정을 포함한 전체 데이터:** 앱과 전용 Edge를 종료한 뒤 `%APPDATA%\kream-helper` 폴더 삭제

프로그램만 제거하면 업데이트와 재설치를 위해 로컬 데이터가 유지됩니다. 공용 컴퓨터를 반납하거나 다른 사람에게 넘길 때는 전체 데이터 폴더도 삭제하세요.

자세한 서버 처리 내용은 [개인정보 처리방침](/privacy)을 확인하세요.
