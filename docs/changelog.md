# 변경 내역

## 0.1.3

- 제품명을 `KREAM Helper`로 확정하고 새 아이콘을 적용했습니다.
- 여러 상품 감지 목록과 감지 기록 화면의 문서형 UI를 정리했습니다.
- 상품 검색, 사이즈, 수량과 감지 주기 설정 흐름을 개선했습니다.
- 자동 신청 예상 보증금과 시작 전 확인 절차를 강화했습니다.
- KREAM 로그인 상태를 자동으로 확인하고 로그인 완료 시 Edge 창을 닫도록 개선했습니다.
- 최종 이용약관, 개인정보 처리방침과 GitHub 문의 정보를 추가했습니다.

## 0.1.2

- 모든 사용자용 Windows 설치와 관리자 권한 자동 업데이트를 검증했습니다.
- 앱 업데이트 후 자동 재실행 흐름을 안정화했습니다.

## 0.1.1

- 자동 업데이트 회귀 검증을 위한 배포 버전을 구성했습니다.

## 0.1.0

- KREAM 로그인, 상품 검색, 다중 감지와 선택적 자동 신청 기능을 포함한 첫 Windows 테스트 버전입니다.
